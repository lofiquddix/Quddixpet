## Packages
framer-motion | Essential for complex, physics-based pet movements, jumps, and UI animations
lucide-react | Used for pet icons (Cat, Dog, Rabbit, etc.) and UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
WebSocket connects to /ws path for real-time game state updates.
Transparent mode for OBS can be activated via URL parameter: ?transparent=true
App is localized in Russian as requested for the QUDDIX Twitch channel.
