## Packages
framer-motion | Essential for smooth, complex pet animations, floating text, and dynamic UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts
lucide-react | Already installed, but crucial for pet icons and UI

## Notes
- WebSocket connection will be established at `/ws` for real-time game updates.
- Transparent mode for OBS can be triggered via `?transparent=true` in the URL.
- No stock images used; relying on pure CSS gradients, SVG patterns, and Lucide icons to ensure a clean, fast-loading, customizable overlay.
